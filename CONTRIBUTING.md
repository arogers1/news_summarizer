# Contributing to News Summarizer

Thank you for your interest in contributing to the News Summarizer project!

## How to Contribute

### Reporting Issues

If you find a bug or have a suggestion:

1. Check if the issue already exists in the [Issues](https://github.com/arogers1/news_summarizer/issues) section
2. If not, create a new issue with:
   - A clear, descriptive title
   - Detailed description of the issue or suggestion
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Your environment (OS, Python version, etc.)

### Contributing Code

1. **Fork the repository**
   ```bash
   git fork https://github.com/arogers1/news_summarizer.git
   ```

2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make your changes**
   - Follow the existing code style
   - Add comments for complex logic
   - Update documentation if needed

4. **Test your changes**
   - Ensure the code runs without errors
   - Test with different configurations
   - Verify nothing breaks existing functionality

5. **Commit your changes**
   ```bash
   git commit -m "Add feature: your feature description"
   ```

6. **Push to your fork**
   ```bash
   git push origin feature/your-feature-name
   ```

7. **Create a Pull Request**
   - Go to the original repository
   - Click "New Pull Request"
   - Describe your changes clearly
   - Reference any related issues

## Code Style

- Follow PEP 8 for Python code
- Use meaningful variable and function names
- Add docstrings to functions and classes
- Keep functions focused and concise

## Areas for Contribution

We welcome contributions in these areas:

- **Bug fixes**: Help us squash bugs!
- **New features**: Email delivery, multi-source support, etc.
- **Documentation**: Improve guides, add examples
- **Testing**: Add unit tests, integration tests
- **Performance**: Optimize scraping and processing
- **UI/UX**: Better output formatting, progress bars

## Questions?

Feel free to open an issue with your question or reach out to the maintainers.
